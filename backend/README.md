# NODE JS CORE

## config proyect

- Create .env file
`cat .env.example > .env`

- Install nodejs
`16.13.0`

- Install node modules - use npm
`npm install`

- Config the files of .config folder

- Config you database fav

## Run proyect

- run proyect in mode dev
`npm run dev`

- run proyect in mode production
`npm run start`

- run tests
`npm run test`

- run build of proyect
`npm run build`
